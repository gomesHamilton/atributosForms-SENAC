Action

O atributo action define o local, por meio de uma URL, ao qual serão enviados todos os dados recolhidos do formulário.

Method

O atributo method define define o metódo HTTP que o formulário em HTML irá lidar com os dados recebidos. O navegador envia uma requisição por meio da URL sempre que você utiliza algum recurso da web.

Temos os métodos:

  • get
    • Um formulário enviado pelo método get informa os dados, reconhecidos pelo servidor, por meio da URL. Exemplo: url?nome=valor&idade=value. Com isso, esse método não é recomendado para compartilhamento de dados confidenciais, uma vez que essas informações ficarão visíveis na URL da página.

  • post
    • O formulário enviado por esse método irá enviar os dados ao servidor e solicitar o retorno de um resultado adequado. Diferentemente do get, o post não mostra os dados em seu URL.

  • dialog
    • É usada para definir uma caixa de diálogo, facilitando a criação de um modal na página.

Autocomplete

O atributo autocomplete habilita ou desabilita o autopreenchimento dos elementos form e input.

Enctype

O atributo enctype especifica a forma de codificação que os dados serão enviados ao servidor.

  • application/x-www-form-urlencoded
    • É o valor padrão. Ele codifica todos os caracteres antes de serem enviados ao servidor. Ele converte espaços em símbolos + e caracteres especiais em seu valor hexadecimal.
  • multipart/form-data
    • Este valor não codifica nenhum caractere.
  • text/plain
    • Este valor converte espaços em símbolos +, mas os caracteres especiais não são convertidos.

Target

O atributo Target serve pra especificar ONDE abrir o link:

_blank: abre o link em uma nova janela;
_self: o mesmo quadro em que estiver o link;
_top: abre as páginas na janela do navegador, desconsiderando qualquer frame; é como "limpar" os frames;
_parent: a página pai, a que contém o frame que contém a página atual.

Name

O atributo HTML <"input"> name é usado para especificar um nome para um elemento <"input">.
É usado para fazer referência aos dados do formulário após o envio do formulário ou para fazer referência ao elemento em um JavaScript.

Novalidate

O atributo HTML novalidate é usado para especificar que os dados do formulário não devem ser validados ao enviar o formulário.
É um atributo booleano.
